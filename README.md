# Desafio iPhone - DIO

Projeto desenvolvido em Java para representar os papéis do iPhone utilizando Programação Orientada a Objetos e interfaces.

## Funcionalidades

O iPhone foi modelado com base em três principais funções:

### 🎵 Reprodutor Musical
- tocar música
- pausar música
- selecionar música

### 📞 Aparelho Telefônico
- ligar
- atender
- iniciar correio de voz

### 🌐 Navegador na Internet
- exibir página
- adicionar nova aba
- atualizar página
